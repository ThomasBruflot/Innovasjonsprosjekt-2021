# Innovasjonsprosjekt-2021

All kode som blir brukt til innovasjonsprosjektet for ELSYS 2021. Prosjektet er laget av gruppe 1.

Merk: filene i mappen "Arduino" er kun for testing av de ulike delsystemene, så ingen av disse filene skal lastes opp på mikrokontrollerne så framt det ikke skal testes noe.
